About This Admin

BLOK. Admin Panel v1.0 — Client-side admin interface for the BLOK. WebDev Studio site.

• Data Storage: All data is persisted via Netlify Blobs.
• Authentication: Credentials are stored in a gitignored credentials.json file — never in the repo.
• Leads: Contact form submissions from the live site are captured here.
• Portfolio: Items managed here are independent from the hardcoded items in index.html.
