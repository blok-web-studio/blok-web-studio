About This Admin

BLOK. Admin Panel v1.0 — Client-side admin interface for the BLOK. WebDev Studio site.

• Data Storage: All data is stored in your browser's localStorage. Nothing is sent to any server.
• Authentication: Credentials are stored in a gitignored credentials.json file — never in the repo.
• Leads: Contact form submissions from the live site are captured here. To capture real submissions, the site's contact form must post to admin/leads/import or use the localStorage API.
• Portfolio: Items managed here are independent from the hardcoded items in index.html. Update the HTML to reflect changes.

STRUCTURAL INTEGRITY: 98.7% · BUILT WITH BLOK.
